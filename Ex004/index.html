Operadores Aritméticos

JavaScript

Sinais dos Operadores:

  • + ----- Adição (5+2 = 7)
  • - ----- Subtração (5-2 = 3)
  • * ----- Multiplicação (5*2 = 10)
  • / ----- Divisão(real) (5/2 = 2.5)
  • % ----- Resto da divisão(inteiro) (5%2 = 1)
  • ** ----- Potenciação (5**2 = 25)
